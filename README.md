# Guru99 – Manual Testing Project

## Project Overview
This project demonstrates **Manual Testing** techniques using **Jira & Zephyr**. 
The focus was on tracking **User Stories**, documenting test executions, and capturing detailed screenshots.

## Tools Used
- Jira
- Zephyr (screenshots used for test execution tracking)
- MS Word / PDF for documentation

## What I Did
- Collected **User Stories** and **Pages** into Word documents
- Captured **Screenshots** of Jira boards and test execution
- Maintained a **project link** and notes for reference
- Structured all documentation for easy understanding and sharing

## Folder Structure
- `Documentation/` → User Stories, Pages, Screenshots
- `LinksAndNotes/` → Project links or notes
- `README.md` → Project description
